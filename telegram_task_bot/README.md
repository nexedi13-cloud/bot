# Telegram Task Bot

Этот бот позволяет управлять задачами в Telegram чате вашей команды. Он может создавать задачи, отслеживать их статус и напоминать о дедлайнах.

## Функциональность

- Создание задач из сообщений в чате
- Отслеживание статуса задач
- Назначение задач пользователям
- Просмотр всех задач и задач конкретного пользователя
- Установка дедлайнов для задач

## Установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Замените `YOUR_TELEGRAM_BOT_TOKEN` в файле `bot.py` на токен вашего Telegram бота (получите его у @BotFather)

3. Запустите бота:
   ```bash
   python bot.py
   ```

## Использование

### Создание задачи
1. Отправьте сообщение с текстом задачи в чат
2. Ответьте на это сообщение, упомяните бота и напишите "Запомни" или "Создай задачу"
3. При необходимости укажите дату выполнения в формате YYYY-MM-DD

Пример:
```
User1: Нужно подготовить презентацию для встречи
User2: @task_bot Запомни
```

### Обновление статуса задачи
Ответьте на сообщение с задачей и напишите один из следующих статусов:
- "Готово" / "Сделано" / "Выполнено" - задача выполнена
- "В работе" / "Работаю" / "В процессе" - задача в процессе выполнения
- "Ожидает" / "Пауза" / "Приостановлено" - задача ожидает

### Команды
- `/start` - начать работу с ботом
- `/help` - получить справку
- `/tasks` - посмотреть все активные задачи
- `/mytasks` - посмотреть задачи, назначенные вам
- "все задачи" / "задачи" - посмотреть все активные задачи (альтернатива команде /tasks)
- "мои задачи" - посмотреть задачи, назначенные вам (альтернатива команде /mytasks)

## Технические детали

- Бот использует SQLite базу данных для хранения задач
- Задачи связаны с сообщениями в чате для удобства отслеживания
- Статусы задач: pending (ожидает), in progress (в работе), done (выполнено), waiting (ожидает)
